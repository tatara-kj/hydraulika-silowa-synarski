# MEDIA READINESS

## BLOCKED

Data werdyktu: **2026-07-23**

Aktualizacja wdrożeniowa: kompletna wersja bez mediów została zbudowana do lokalnej akceptacji UX i treści. Status publikacyjny pozostaje bez zmian.

### Dlaczego

- Oficjalna wizytówka Google jest potwierdzona, ale zawiera tylko mapę dojazdu i zdjęcie drogi.
- Żaden dostępny materiał nie pokazuje warsztatu, sprzętu, pracownika, siłownika, procesu naprawy ani rezultatu.
- Nie ma kompletnego przypadku „problem → diagnoza → naprawa → rezultat”.
- Nie ma podstawy prawnej do ponownej publikacji materiałów z Google; oba wymagają zgody właściciela.
- Nie odnaleziono potwierdzonych oficjalnych profili społecznościowych z dodatkowymi mediami.

### Warunki zmiany na READY

Status można zmienić wyłącznie po spełnieniu wszystkich warunków:

1. właściciel przekaże lub wskaże autentyczne materiały firmy;
2. potwierdzi autorstwo i zgodę na wykorzystanie w nowej stronie;
3. materiały obejmą co najmniej warsztat/zaplecze oraz 3 prawdziwe realizacje;
4. każda realizacja będzie miała bezpieczny, niepoufny opis;
5. pliki będą wystarczającej jakości do wersji desktop i mobile;
6. lista usług widocznych w materiałach zostanie potwierdzona przez właściciela.

### Konsekwencja projektowa

Nie budować finalnego hero fotograficznego, galerii realizacji ani case studies w Prompt 2. Nie zastępować braków stockami ani generowanymi „realizacjami”. Można przygotować wyłącznie szkielet komponentów bez publicznych fikcyjnych treści.
